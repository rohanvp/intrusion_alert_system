# Solar Powered Wireless Intrusion Alert System
## Components Used:
- Arduino Mega 2560
- Ultrasonic Distance Sensor
- HC12 RF Transceiver
- LED and Resistor
- Solar Panel ( 3V 100mA )
- Buzzer
- USB Voltage Booster
- TP4056 Charging Module
- ICR 18650 Battery
